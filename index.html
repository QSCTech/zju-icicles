еүҚиЁҖ

жқҘеҲ°дёҖжүҖеӨ§еӯҰпјҢд»Һз¬¬дёҖж¬ЎжҺҘи§Ұи®ёеӨҡиҜҫпјҢзӣҙеҲ°дёҖй—ЁдёҖй—Ёе®ҢжҲҗпјҢиҝҷдёӘиҝҮзЁӢдёӯжҲ‘д»¬ж—¶еёёж”¶йӣҶиө·и®ёеӨҡиө„ж–ҷе’Ңжғ…жҠҘгҖӮ

жңүдәӣжҳҜйңҖиҰҒеңЁзҪ‘дёҠжҗңзҙўзҡ„з”өеӯҗд№ҰпјҢжҜҸж¬Ўи§ҒеҲ°дёҖй—Ёж–°иҜҫзЁӢпјҢGoogle дёҖдёӢж•ҷжқҗеҗҚз§°пјҢжңүзҡ„еҸҜд»Ҙз«ӢеҚіжүҫеҲ°пјҢжңүзҡ„еҚҙжҳҜиҰҒиҠұиҙ№и®ёеӨҡзңјеҠӣпјӣжңүдәӣжҳҜеҺҶе№ҙиҜ•еҚ·жҲ–иҖ… A4 зәёпјҢеүҚдәәзІҫеҝғж”¶йӣҶеҲ¶дҪңпјҢжҠұзқҖиғҪеҜ№д»–дәәжңүз”Ёзҡ„жғіжі•е…¬ејҖпјҢеҚҙйңҖиҰҒеңЁеҗ„дёӘзҫӨжҲ–иҖ… CC98 дёӯж‘ёзҙўд»ҘиҮідәҺд»ҺеӯҰй•ҝжүӢдёӯд»Јд»Јзӣёдј пјӣжңүдәӣжҳҜдёҠе®ҢдёҖй—ЁиҜҫжүҚжҒҚз„¶йўҶжӮҹзҡ„жҠҖе·§пјҢеҺҹжқҘиҝҷй—ЁиҜҫйҮҚзӮ№еҰӮжӯӨпјҢеҪ“еҲқжң¬еҸҜд»ҘжӣҙиҪ»жқҫең°е®ҢжҲҗеҫ—жӣҙеҘҪвҖҰвҖҰ

жҲ‘д№ҹжӣҫеҫҲеҠӘеҠӣең°ж”¶йӣҶеҗ„з§ҚиҜҫзЁӢиө„ж–ҷпјҢдҪҶеҲ°жңҖеҗҺпјҢжҹҗдәӣйҮҚиҰҒдҝЎжҒҜзҡ„еҫ—еҲ°еҚҙеҫҖеҫҖдҫқз„¶жҳҜзәҜеұһеҒ¶з„¶гҖӮиҝҷз§ҚзҠ¶жҖҒж—¶еёёд»ӨжҲ‘ж„ҹеҲ°еҗҺжҖ•дёҺдёҚе®үгҖӮжҲ‘д№ҹжӣҫеңЁиҜҫзЁӢз»“жқҹеҗҺз»ҲдәҺжңүдәҶдәӣи®ёж–№жі•дёҺжҖ»з»“пјҢдҪҶиҝҷдәӣжғіжі•ж— еӨ„иҜүиҜҙпјҢжңҖз»ҲеҸӘиғҪжҠҠиҠұиҙ№ж—¶й—ҙдёҺзІҫеҠӣжүҚжҚўжқҘзҡ„з»ҸйӘҢиҖ—ж•ЈеңЁдәҶжј«жј«зҡ„йҒ—еҝҳд№ӢдёӯгҖӮ

жҲ‘дёәиҝҷдёҖе№ҙдёҖе№ҙпјҢиҝҷд№ҲеӨҡдәәеӯӨеҶӣеҘӢжҲҳзҡ„йҮҚеӨҚеҠіеҠЁж„ҹеҲ°дёҚе№ігҖӮ

жқҘиҮӘеӨ§е®¶зҡ„е…іжіЁгҖҒз»ҙжҠӨе’ҢиҙЎзҢ®пјҢжүҚжҳҜи®©иҝҷдёӘжөҷжұҹеӨ§еӯҰзӢ¬жңүзҡ„ж”»з•Ҙжң¬з»§з»ӯеӯҳеңЁзҡ„еҠЁеҠӣ~

ж“ҚдҪңж–№жі•

жҸҗдәӨ PRпјҡFork жң¬йЎ№зӣ®пјҢз„¶еҗҺеңЁ GitHub зҪ‘йЎөз«ҜзӮ№еҮ» Upload File дёҠдј ж–Үд»¶пјҢеҸ‘иө· PR еҚіеҸҜгҖӮз•ҷж„ҸдёҖдёӢйЎ№зӣ®зҡ„ж–Үд»¶з»„з»Үе–”гҖӮ

жҲ–иҖ…д№ҹеҸҜд»ҘзӣҙжҺҘйҷ„еҠ еңЁ Issue дёӯпјҢз”ұз»ҙжҠӨиҖ…иҝӣиЎҢж·»еҠ гҖӮ

еҜ№дәҺж•ҷеёҲзҡ„иҜ„д»·иҜ·дёҖеҫӢдҪҝз”Ёе§“еҗҚжӢјйҹійҰ–еӯ—жҜҚзј©еҶҷпјӣиҮідәҺж•ҷеёҲжҸҗдҫӣзҡ„иҜҫд»¶е°ұдёҚз”ЁдёҠдј дәҶпјҢеӣ дёәжҜҸе№ҙиҜҙдёҚе®ҡдјҡжңүжӣҙж–°зҡ„еҳӣгҖӮ

з”ұдәҺжң¬йЎ№зӣ®дҪ“з§ҜеҫҲеӨ§пјҢж•…еҸҜд»ҘйҮҮз”ЁеңЁGithub Webз«ҜзӣҙжҺҘдёҠдј зҡ„ж–№ејҸпјҢе…·дҪ“ж“ҚдҪңеҰӮдёӢпјҡ

  1. йҰ–е…ҲForkжң¬йЎ№зӣ®

  2. дёҠдј ж–Үд»¶еҲ°е·Іжңүж–Үд»¶еӨ№пјҡжү“ејҖеҜ№еә”ж–Үд»¶еӨ№пјҢзӮ№еҮ»з»ҝиүІDownloadжҢүй’®ж—Ғзҡ„uploadпјҢдёҠдј дҪ зҡ„ж–Үд»¶гҖӮ

  3. дёҠдј ж–Үд»¶еҲ°ж–°ж–Үд»¶еӨ№пјҡжү“ејҖд»»ж„Ҹж–Үд»¶еӨ№пјҢзӮ№еҮ»з»ҝиүІDownloadжҢүй’®ж—Ғзҡ„uploadпјҢжҠҠжөҸи§ҲеҷЁең°еқҖж Ҹдёӯж–Үд»¶еӨ№еҗҚз§°ж”№дёәдҪ жғіиҰҒж–°е»әзҡ„ж–Үд»¶еӨ№еҗҚз§°пјҢз„¶еҗҺеӣһиҪҰпјҢдёҠдј дҪ зҡ„ж–Үд»¶гҖӮ

жҸҗйҶ’

жңүдәӣжңӢеҸӢеңЁжҸҗдәӨ PR зҡ„ж—¶еҖҷеҸҜиғҪдјҡжіЁж„ҸеҲ°иҮӘе·ұзҡ„ Fork е’ҢжҲ‘д»¬зҡ„дё»еҲҶж”Ҝжңүж•°еҚҒз”ҡиҮідёҠзҷҫдёӘдёҚеҗҢзҡ„ commit гҖӮеҰӮжһңеҮәзҺ°иҝҷз§Қжғ…еҶөпјҢеҸҜд»ҘиҖғиҷ‘д»ҘдёӢдёӨз§Қи§ЈеҶіж–№жЎҲпјҡ

  1. еҰӮжһңеҜ№gitдёҚеӨӘзҶҹжӮүпјҢе»әи®®пјҲеңЁеӨҮд»Ҫе®ҢжҲҗеҗҺпјүе…ҲеҲ йҷӨдҪ зҡ„йЎ№зӣ®пјҢйҮҚж–° fork гҖҒдёҠдј е№¶йҮҚж–°жҸҗдәӨ PR гҖӮ
  2. еҰӮжһңеҜ№gitеҸҠе…¶е·ҘдҪңеҺҹзҗҶиҫғдёәзҶҹжӮүпјҲиҖҢдё”ж„ҝж„ҸиҠұиҙ№ж—¶й—ҙе’ҢжөҒйҮҸжҠҳи…ҫпјүпјҢеҸҜд»Ҙе°қиҜ•еңЁ fork еҮәзҡ„йЎ№зӣ®дёҠиҝӣиЎҢ rebase д»Ҙж¶ҲйҷӨдёҺдё»еҲҶж”ҜеңЁеҺҶеҸІдёҠзҡ„еҶІзӘҒгҖӮ

иӯҰе‘Ҡ

дёӢеҲ—еҶ…е®№дёәдёҚйҖӮеҗҲдёҠдј зҡ„еҶ…е®№гҖӮеҰӮжһңдҪ и®Өдёәзјәе°‘иҝҷдәӣиө„ж–ҷе°ҶдјҡеҪұе“Қиө„жәҗзҡ„е®Ңж•ҙжҖ§пјҢиҜ·дјҳе…ҲиҖғиҷ‘ж”ҫеңЁж ЎеҶ…иө„жәҗе№іеҸ°пјҢжҲ–иҒ”зі»дҪ зҡ„ж•ҷеёҲе№¶з”ұж•ҷеёҲеҸ‘еёғгҖӮе»әи®®дҪ ж’°еҶҷдёҖдёӘ README ж–ҮжЎЈе№¶ж”ҫзҪ®дёҖдәӣй“ҫжҺҘжҲ–жҢҮеј•ж–Үеӯ—жқҘеё®еҠ©жүҫеҲ°иҝҷдәӣиө„жәҗгҖӮ

  • зӣ—зүҲз”өеӯҗд№Ұ/д»ҳиҙ№з”өеӯҗд№Ұ
  • зӣ—зүҲ/з ҙи§ЈзүҲ/з»ҝиүІзүҲд»ҳиҙ№иҪҜд»¶еҸҠе…¶е®үиЈ…еҢ…
  • иҜҫзЁӢ/ж•ҷеёҲдё»йЎөдёҠеҲ—еҮәзҡ„еҶ…е®№пјҲиҜ·еңЁиҺ·еҫ—ж•ҷеёҲи®ёеҸҜеҗҺдёҠдј пјү

еҰӮжһңдҪ и®Өдёәжң¬д»“еә“зҡ„дёҖдәӣж–Үд»¶дҫөзҠҜдәҶжӮЁзҡ„жқғзӣҠпјҢиҜ· еҗ‘жҲ‘д»¬еҸ‘йҖҒйӮ®д»¶ гҖӮжҲ‘д»¬е°Ҷдјҡд»Һд»“еә“дёӯеҪ»еә•жё…йҷӨиҝҷдәӣж–Үд»¶гҖӮ

и®ёеҸҜ

з”ұиҙЎзҢ®иҖ…зј–еҶҷйғЁеҲҶзҡ„и®ёеҸҜеҰӮдёӢпјҡ

CC-BY-NC-SAпјҡзҪІеҗҚ-йқһе•ҶдёҡжҖ§дҪҝз”Ё-зӣёеҗҢж–№ејҸе…ұдә«

иө„ж–ҷд»…дҫӣеҸӮиҖғпјҢиҜ·иҮӘе·ұеҲӨж–ӯе…¶йҖӮз”ЁжҖ§гҖӮ

е…¶д»–йғЁеҲҶзҡ„зүҲжқғеҪ’еұһдәҺе…¶еҗ„иҮӘзҡ„дҪңиҖ…гҖӮ
