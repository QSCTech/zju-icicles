# 算法分析

## 渐进符号

大O符号 T(N) = O(f(N)) T(N) <= c * f(N)

大Ω符号 T(N) = Ω(f(N)) T(N) >= c * f(N)

大Θ符号 T(N) = Θ(f(N)) T(N) = O(f(N)) && T(N) = Ω(f(N))

小o符号 T(N) = o(f(N)) T(N) =

* T1(N) + T2(N) = max(O(f(N)), O(g(N))

* T1(N) * T2(N) = O(f(N) * g(N))

* 若 T(N) 为 k 次多项式，则 T(N) = Θ(N ^ k)

* log(k, N) = O(N)

## 最大子列问题

线性算法：从头向尾加和，若和小于零则从下一个位置重新开始。
